# Crawl Data Crypto with React JS, Material UI and Chart JS

## Built with

- [React JS](https://reactjs.org/)
- [Material UI](https://v4.mui.com/)
- [Chart JS](https://reactchartjs.github.io/react-chartjs-2/#/)

## Installation
Step 1: Install
``
yarn || npm install
``

Step 2: Run local
``
yarn start || npm run start
``
